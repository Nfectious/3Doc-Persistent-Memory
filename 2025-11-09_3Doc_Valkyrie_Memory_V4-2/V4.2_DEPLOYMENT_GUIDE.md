## 🚀 DEPLOYMENT

### Step 1: Extract Package

```bash
tar -xzf v4.2_paste_nextcloud.tar.gz
```

### Step 2: Fix PHP Upload Limits

```bash
# Edit PHP config
sudo nano /etc/php/7.4/fpm/php.ini

# Find and update:
upload_max_filesize = 10M
post_max_size = 10M
max_execution_time = 60

# Save and restart
sudo systemctl restart php7.4-fpm
```

### Step 3: Fix Permissions

```bash
# Inbox directory
sudo mkdir -p /opt/valkyrie/memory/_incoming
sudo chown -R www-data:valkyrie /opt/valkyrie/memory/_incoming/
sudo chmod -R 775 /opt/valkyrie/memory/_incoming/

# All memory
sudo chown -R valkyrie:valkyrie /opt/valkyrie/memory/
sudo chmod -R 755 /opt/valkyrie/memory/
```

### Step 4: Deploy New Files

```bash
# Backup current version
sudo cp /var/www/valkyrie/index.html /var/www/valkyrie/index_v4.1_backup.html
sudo cp /var/www/valkyrie/api.php /var/www/valkyrie/api_v4.1_backup.php

# Deploy V4.2
sudo cp index_v4.2_paste.html /var/www/valkyrie/index.html
sudo cp api_v4.2_paste.php /var/www/valkyrie/api.php

# Restart services
sudo systemctl restart nginx php7.4-fpm
```

### Step 5: Test Upload/Paste

1. Open: `http://aimem.bsapservices.com`
2. Click "📤 Upload Export" tab
3. Try **Paste Method**:
   - Click "📝 Paste Text"
   - Paste session export
   - Select project (or auto-detect)
   - Click "Process Export"
4. Try **File Method**:
   - Click "📁 Upload File"
   - Drop or select file
   - Should work now!

## 🐳 NEXTCLOUD INTEGRATION (Optional)

Follow complete guide in `NEXTCLOUD_DOCKER_INTEGRATION.md`

### Quick Setup

```bash
# 1. Mount in docker-compose.yml
volumes:
  - /opt/valkyrie/memory:/var/www/html/data/[USERNAME]/files/Valkyrie:rw

# 2. Fix permissions
sudo chown -R 33:33 /opt/valkyrie/memory/
sudo chmod -R 775 /opt/valkyrie/memory/

# 3. Restart Nextcloud
docker-compose restart nextcloud

# 4. Rescan files
docker exec -u www-data nextcloud php occ files:scan [USERNAME]

# 5. Install watcher (see full guide)
sudo nano /opt/valkyrie/inbox_watcher.sh
# ... copy script from guide ...
sudo systemctl enable valkyrie-watcher
sudo systemctl start valkyrie-watcher
```

## 🐛 TROUBLESHOOTING

### File Upload Still Fails

```bash
# Check PHP error log
sudo tail -f /var/log/php7.4-fpm.log

# Check nginx error log
sudo tail -f /var/log/nginx/error.log

# Verify permissions
ls -la /opt/valkyrie/memory/_incoming/

# Should show:
# drwxrwxr-x valkyrie valkyrie _incoming/

# Test API directly
curl -X POST http://localhost/valkyrie/api.php?action=list_projects
```

### Paste Doesn't Work

```bash
# Check API endpoint
curl -X POST http://localhost/valkyrie/api.php?action=process_export \
  -H "Content-Type: application/json" \
  -d '{"export_text":"test","project":"global"}'

# Should return JSON response

# Check browser console (F12)
# Look for JavaScript errors
```

### Nextcloud Files Not Showing

```bash
# Force rescan
docker exec -u www-data nextcloud php occ files:scan --all

# Check mount
docker exec nextcloud ls -la /var/www/html/data/[USERNAME]/files/

# Restart container
docker restart nextcloud
```

### Watcher Not Processing

```bash
# Check service
sudo systemctl status valkyrie-watcher

# View logs
sudo tail -f /var/log/valkyrie_watcher.log

# Restart
sudo systemctl restart valkyrie-watcher

# Test manually
sudo -u valkyrie /opt/valkyrie/inbox_watcher.sh
```
